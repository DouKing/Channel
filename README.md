## Channel

A common function development kit for iOS.

### Installation

- **Using Swift Package Manager**

  ```swift
  import PackageDescription

  let package = Package(
    name: "MyAwesomeApp",
    dependencies: [
      .package(url: "https://github.com/douking/Channel", from: "0.1.0"),
    ]
  )
  ```
